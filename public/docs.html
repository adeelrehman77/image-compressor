NexusCompress

Professional client-side image optimization. All compression runs in your browser — nothing is uploaded to a server.

Features

  • Presets for web, email, social, and archive workflows
  • WebP, AVIF, JPEG, PNG output
  • Resize, quality control, and target file size
  • Batch processing, folder import, ZIP export
  • Before/after comparison slider

CLI

npm run compress -- ./photos --preset web --out ./optimized

Deploy

Built for Cloudflare Pages. See DEPLOY.md in the repository.

Privacy

Images are processed locally using Web Workers and OffscreenCanvas. Metadata is stripped when images are re-encoded.
