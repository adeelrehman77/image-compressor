NexusCompress · Best image format for UAE government portals

Uploading documents to ICA, MOHRE, RTA, and Emirates ID portals often fails because of the wrong format or file size. Here is what each portal accepts and how to prepare images without uploading them to a cloud service.

The short answer: always use JPEG

UAE government portals universally accept JPEG. PNG may work on some portals but is rarely preferred. WebP and AVIF are not accepted on UAE government portals — always convert to JPEG before uploading.

Why PNG often gets rejected

PNG files are lossless and cannot be compressed to a target file size the way JPEG can. A scanned ID in PNG format is often 1–3 MB — well above the 200 KB portal limit. Save scans as JPEG for portal uploads.

How to prepare your image with NexusCompress

  1. Open NexusCompress — your files never leave your device.
  2. Under UAE Portal Presets, select the matching portal (e.g. Emirates ID / passport copy — 200 KB JPEG).
  3. Drop your image and download the compressed file.
  4. Enable target file size — the tool lowers quality step-by-step until the file fits.

Tip: Scan documents at 150–200 DPI rather than 300 DPI. Higher resolution creates a larger file with no benefit for portal uploads.

Frequently asked questions

What image format does ICA accept for UAE visa uploads?

ICA accepts JPEG images under 200 KB. Use NexusCompress with the ICA / visa upload preset to hit the right size automatically.

Can I upload WebP to UAE government portals?

No. UAE government portals do not accept WebP or AVIF. Always convert to JPEG before uploading.

What is the maximum file size for Emirates ID portal uploads?

Emirates ID and ICA portals typically require images under 200 KB in JPEG format.

Is it safe to compress my passport scan online?

With NexusCompress, yes — all processing happens in your browser. Passport and ID scans are never sent to any server.
