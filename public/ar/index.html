      NexusCompress — ضاغط صور مجاني على الإنترنت

      تصغير الملفات في ثوانٍ

      NexusCompress ضاغط صور مجاني يصغّر ملفات JPEG وPNG وWebP وAVIF دون إرسالها إلى السحابة. اسحب الصور، اضبط الجودة والأبعاد، ثم حمّل النتائج — مثالي للمواقع والبريد ووسائل التواصل.

      لا شيء يغادر جهازك

      على عكس الأدوات التي ترفع ملفاتك، كل المعالجة تتم محلياً في متصفحك — سرعة أعلى، خصوصية أقوى، ولا مخاطرة من بقاء صورك على خوادم طرف ثالث.

      جاهز لسير عمل حقيقي

      ضغط دفعات، حدود 200/500 ك.ب لبوابات الإمارات، مقارنة قبل/بعد، وتصدير ZIP — بدون حساب وبدون تثبيت.

      لماذا ضاغط صور يعمل في المتصفح؟
      • بدون رفع — الصور لا تغادر جهازك
      • معالجة دفعات — ضغط عدة ملفات وتحميل ZIP
      • صيغ حديثة — WebP وAVIF لملفات أصغر بجودة مماثلة
      • مقارنة بصرية — شريط قبل/بعد قبل الحفظ

      أدوات وسائط مجانية (نفس الموقع، بدون رفع)

      استخدم التبويبات أعلاه، أو انتقل مباشرة:

      ضاغط الصور صور إلى PDF دمج وتقسيم PDF تحسين SVG استوديو صور الهوية محوّل HEIC محوّل الصيغ

      رفع بوابات حكومة الإمارات (حدود 200 / 500 ك.ب)

      استخدم إعدادات بوابات الإمارات في تبويب استوديو صور الهوية لقص الهوية الإماراتية وICA. إعدادات MOHRE وRTA والبوابة العامة تفتح ضاغط الصور بحجم ملف مناسب — بدون رفع، فتبقى نسخ الجواز والتأشيرة على جهازك.

      تغيير الحجم، النسبة، والعلامة المائية

      غيّر الحجم بالنسبة المئوية أو اقص إلى 16:9 أو 4:3 أو مربع. أضف علامة مائية نصية أو شعار لدفعة كاملة ثم حمّل ZIP مع شريط تقدم.

      أدلة الضغط

      • ضغط صور بوابات الإمارات (200/500 ك.ب)
      • أفضل صيغة صورة لبوابات حكومة الإمارات
      • تغيير حجم صورة تأشيرة الإمارات (ICA)
      • ضغط صورة لبوابة MOHRE / العمل
      • ضغط صورة لواتساب
      • تصغير صورة لمرفقات البريد
      • صور عقارات (بيوت، دوبيزل)
      • تصغير صورة لمواقع WordPress
      • ضغط JPEG دون فقدان الجودة
      • JPEG مقابل WebP — أيهما تختار؟
      • تغيير حجم صور إنستغرام
      • جميع الأدلة

      الأسئلة الشائعة

      كيف أضغط JPEG دون فقدان الجودة؟

      استخدم جودة بين 75% و85% للصور. للويب، حوّل إلى WebP أو AVIF — غالباً ملفات أصغر بنفس الجودة البصرية. راجع دليل ضغط JPEG أو دليل بوابات الإمارات. استخدم شريط قبل/بعد في NexusCompress قبل التحميل.

      هل من الآمن رفع صوري هنا؟

      صورك لا تُرفع أبداً. NexusCompress يعالج الصور بالكامل داخل متصفحك عبر Web Workers وOffscreenCanvas. الملفات تبقى على جهازك ولا تُرسل إلى خوادمنا — مناسب للقطات الحساسة وأعمال العملاء والهوية والجواز.

      ما أفضل صيغة لتصغير حجم الصورة؟

      لمعظم المواقع، WebP توازن ممتاز (قارن JPEG وWebP). AVIF أصغر حيث مدعوم (AVIF مقابل WebP). استخدم JPEG للبريد؛ PNG للشفافية.

      هل يمكن ضغط عدة صور دفعة واحدة؟

      نعم. اسحب عدة ملفات أو استورد مجلداً كاملاً. NexusCompress يصف كل صورة ويعرض التقدم والتوفير لكل ملف، ثم يمكنك تحميل الكل كملف ZIP واحد.

      هل الضغط يزيل EXIF أو موقع GPS؟

      إعادة الترميز تزيل البيانات الوصفية مثل موقع GPS وطراز الجهاز والطوابع الزمنية. NexusCompress يصحّح اتجاه EXIF لتظهر صور الهاتف بالاتجاه الصحيح — مثالي للخصوصية عند المشاركة.

      هل الأداة مجانية فعلاً؟

      نعم. الضغط الأساسي مجاني بدون حساب. الأداة تعمل في متصفحك دون رسوم API لكل صورة — الإعلانات الاختيارية تدعم الاستضافة فقط.
